# v0.4.0 -- Content Restructuring: Smoother Learning Curve

**Release date:** 2026-03-03
**Tag:** v0.4.0

## Summary

Complete restructuring of all 30 lessons to create a smoother learning curve. Vowels are now introduced alongside consonants from R01, so students read real Thai syllables from their very first lesson. All trivia and cultural questions removed from assessments, replaced with reading-skills exercises. Vocabulary meaning testing deferred until R12 when students have genuine reading fluency.

## Changes

### Content -- Phases 1-3 (R01-R15): Complete rewrite

- R01 now introduces vowel -aa alongside 3 consonants (gaw, jaw, daw), enabling syllable reading from day one
- Vowels interleaved with consonants throughout Phases 1-2 following i+1 principle (3-5 new symbols per lesson, 90% known / 10% new)
- R02 renamed from "More Mid-Class Consonants" to "More Consonants, More Sounds"
- R03 teaches vowel positions (above, below, before, after) instead of rare consonants
- R04 focuses on vowel patterns instead of abstract consonant class theory
- R05 introduces long vs short vowel distinction
- R06-R08 expand consonant set systematically (aspirated, common, more)
- R09 covers special vowels (-am, ai-, short a)
- R10 introduces consonant classes lightly through examples (not abstract theory)
- R11 completes common consonant set with same-sound pairs across classes
- R12 is the first lesson with vocabulary meaning testing (gin, maa, bpai, dii, naam, mai)
- R13 teaches phrase building (adjective-after-noun, verb chaining)
- R14 covers rare consonants (deferred from old R03)
- R15 is comprehensive reading practice with real-world text

### Content -- Phases 4-6 (R16-R30): Trivia removal

- R16: Replaced "Mandarin has 4 tones, Cantonese 6-9" with tone identification exercise
- R20: Replaced "555 means hahaha" with tone analysis exercise
- R21: Replaced "Thai verbs don't conjugate" with sentence reading comprehension
- R23: Replaced "Japanese/Korean use question particles" with question formation exercise
- R26: Replaced "sawasdee adopted in 1940s" with greeting response exercise
- R26: Replaced "When NOT to wai" with greeting exchange reading exercise
- R30: Replaced "mai pen rai reflects Thai tolerance" with sign reading comprehension
