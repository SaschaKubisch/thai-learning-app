# v0.2.0 -- Content Enrichment (Phase 2)

**Release date:** 2026-03-03
**Tag:** v0.2.0

## Summary

Integrated real vocabulary, mini-phrases, and meaning-based exercises into lessons R01-R15. Students now encounter meaningful Thai words from lesson 1 onward, instead of memorizing letters in isolation for 14 lessons before seeing real vocabulary.

## Changes

### Content: Vocabulary integration (R01-R15)

- Added ~38 vocabulary words across 15 lessons, drawn from the existing word bank (`src/content/words.ts`)
- Each lesson now includes:
  - A new `info_card` step introducing 1-4 real Thai words using the lesson's letters
  - A new `rule` step with mini-phrases or simple sentences
  - 1-2 additional `practice` exercises testing word meaning
  - 1-2 revised `assessment` questions mixing letter ID with word meaning
  - Word IDs added to `srsItemIds` for spaced repetition tracking
