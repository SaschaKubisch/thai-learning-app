# v0.5.2 -- Wire Street Thai Env Var Into Deploy Workflow

**Release date:** 2026-03-03
**Tag:** v0.5.2

## Summary

Passes the `VITE_SHOW_STREET_THAI` repository variable into the GitHub Pages deploy workflow so the env var actually takes effect in production builds. Previously the CI build had no `.env` file, so the street thai toggle always rendered regardless of local settings.

## Changes

### Infrastructure

- Updated `.github/workflows/deploy.yml` to pass `VITE_SHOW_STREET_THAI` from `vars.VITE_SHOW_STREET_THAI` into the build step
- Set `VITE_SHOW_STREET_THAI=false` as a GitHub repository variable

## Files modified

- `.github/workflows/deploy.yml`

## Verification

- TypeScript: `npx tsc --noEmit` passes
- Build: `npm run build` succeeds
- Deploy workflow will pick up the repo variable on next push to main
