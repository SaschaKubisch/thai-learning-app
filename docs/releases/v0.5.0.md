# v0.5.0 -- Dark Mode & Street Thai Mode

**Release date:** 2026-03-03
**Tag:** v0.5.0

## Summary

Adds two major features: a dark mode toggle with full theme support across all screens, and Street Thai mode -- an alternative curriculum of 10 crude, funny chapters covering slang, insults, haggling, and other Thai they don't teach in school. Both preferences persist across sessions via localStorage.

## Changes

### Dark Mode

- Sun/moon toggle button in the home screen header
- `dark:` Tailwind classes applied to all 20+ component files with a consistent stone palette
- Theme flash prevention via inline script in `index.html` that reads preferences before first paint
- Tailwind `@custom-variant dark` configured for class-based toggling

### Street Thai Mode

- Fire icon toggle button in the home screen header
- 10 new chapters (S01-S10) covering: crude pronouns, tonal disasters, animal insults, street food ordering, haggling, sarcasm, bar Thai, internet slang, farang problems, and nuclear phrases
- All chapters unlocked with no progression gating (any chapter accessible immediately)
- Street Thai completions do not advance normal curriculum progress (S-prefix guard in `completeRichLesson`)
- Conditional header, phase labels, and progress display when in street mode

### Preferences System

- New `src/engine/preferences.ts` with `loadPreferences()` / `savePreferences()` using localStorage key `thai-app-preferences`
- New `src/hooks/usePreferences.ts` React context/provider following same pattern as `useProgress`
- `switchRichLessons()` function in lesson manager enables hot-swapping the lesson array at render time

### Tests

- 3 new dark mode E2E tests (toggle, persistence, applies to lesson screen)
- 7 new street thai E2E tests (toggle, all chapters clickable, no gating, toggle back, hides buttons, persistence, S01 loads)
