# v0.5.1 -- Gate Street Thai Behind Env Var

**Release date:** 2026-03-03
**Tag:** v0.5.1

## Summary

Adds a `VITE_SHOW_STREET_THAI` environment variable to control whether the Street Thai toggle button is rendered. Set to `false` in `.env` to hide it from the UI. Defaults to shown when unset.

## Changes

### Configuration

- Street Thai toggle button is now conditionally rendered based on `VITE_SHOW_STREET_THAI` env var
- When set to `'false'`, the button is hidden and street thai mode is forced off (even if previously saved in preferences)
- `.env` file is already in `.gitignore`, so the setting is per-deployment
