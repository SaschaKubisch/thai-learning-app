# v1.0.0 -- Public Release

**Release date:** 2026-03-03
**Tag:** v1.0.0

## Summary

First public release of the Thai Learning App. A reading-first Thai language app that teaches all 44 consonants, common vowels, tones, sentences, and dialogues across 30 structured lessons with spaced repetition review. This release cleans up the repository for public sharing by removing internal docs and rewriting the README.
