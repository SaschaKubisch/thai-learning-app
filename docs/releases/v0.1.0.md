# v0.1.0 -- Foundation and Phase 1 UI

**Release date:** 2026-03-03
**Tag:** v0.1.0

## Summary

First tagged release of the Thai Learning App. Includes the complete lesson engine with 30 rich step-based lessons, SRS spaced repetition review, localStorage persistence, GitHub Pages deployment, and Phase 1 UI improvements (Thai font, skill previews, chapter navigation).

## Changes

### Core application (pre-Phase 1)

- 30 rich step-based lessons (R01-R30) across 6 phases: consonants, vowels, reading, tones, sentences, dialogues
- 7 step types: text, info_card, rule, example, table, dialogue, practice
- 4 exercise types: multiple_choice, match, tap_to_reveal, fill_in
- Assessment system with pass/fail (80% threshold) and retry
- SRS spaced repetition engine for review of learned items
- localStorage persistence with v1-to-v2 state migration
- Paiboon+ romanization system with tone diacritics
- Responsive design for mobile, tablet, and desktop
- 39 Playwright E2E tests, 1 Vitest unit test

### Phase 1: UI improvements

- Added Noto Sans Thai from Google Fonts for proper Thai character rendering
- Added `SkillPreview` type and preview data to all 30 lessons
- Rebuilt HomeScreen with all lessons grouped by phase (completed / current / future states)
- Added skill preview modal for future lessons showing what students will learn
- Fixed Vite base path to only apply in production builds (fixes dev server routing)
- Updated E2E tests for new home screen structure

### Infrastructure

- GitHub Pages deployment with Actions workflow
- Playwright configured for headless testing
- Planning skill added to `.claude/skills/`
