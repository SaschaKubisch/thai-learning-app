# v0.3.0 -- Assessment Enhancement (Phase 3)

**Release date:** 2026-03-03
**Tag:** v0.3.0

## Summary

Added 42 new assessment questions across all 30 lessons, focusing on cultural facts, pattern recognition, and practical/contextual scenarios. Also populated srsItemIds for R16-R30 so vocabulary from those lessons is now tracked for spaced repetition. Includes a Thai font fix switching to the traditional looped style.
