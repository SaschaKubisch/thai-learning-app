# Thai Learning App

An interactive app for learning the Thai language. See README.md for project overview.

## Commands

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build
npm run build

# Run tests
npm test

# Lint
npm run lint

# Type check
npx tsc --noEmit
```

## Conventions

- TypeScript strict mode
- React functional components with hooks
- CSS modules or Tailwind for styling
- Tests with Vitest + React Testing Library
